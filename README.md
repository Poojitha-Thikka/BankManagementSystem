# 🏦 Bank Management System

## 📌 Overview
This is a desktop-based Bank Management System developed using Java Swing and MySQL.  
The application simulates real-world banking operations such as account management, transactions, and user authentication.

## 🚀 Features
- 🔐 User Login Authentication
- 💰 Deposit & Withdraw Money
- 📊 Balance Inquiry
- 🧾 Transaction Receipt Generation
- 🔄 Change PIN Functionality
- 🕒 Transaction History with Date & Time

## 🛠️ Technologies Used
- Java (Swing for GUI)
- MySQL (Database)
- JDBC (Database Connectivity)

## 🧠 OOP Concepts Used
- Classes & Objects
- Inheritance (JFrame)
- Interfaces (ActionListener)
- Encapsulation

## 🗄️ Database Details
The system stores:
- User information (PIN, account details)
- Transaction records (type, amount, date)

## ⚙️ How to Run the Project
1. Clone the repository
2. Open the project in IntelliJ IDEA or Eclipse
3. Setup MySQL database
4. Update database credentials in the code
5. Run the main class
